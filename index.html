Resolución de conflictos

GIT es un sistema de control de versiones distribuido, que permite llevar un control de los cambios en el código fuente de un proyecto.

Para resolver conflictos en GIT, se debe realizar un merge de las ramas que contienen los cambios, y luego resolver los conflictos que se presenten.
